# MlChess Bug Report Template

## Preferred Daily Report
- What page or route broke
- What you did
- What you expected
- What actually happened
- Paste the copied debug bundle if available

## Good Minimal Report
`On /#/watch/<match_id> I submitted e2e4 as White. Expected the move to appear immediately. Actual result: the board did not update. Bundle pasted below.`

## Best Short Prompt
`Use the MlChess repo bug workflow and the debug-mlchess-bug skill. Start from this bug bundle or report, reproduce the issue, add a failing regression test if feasible, make the smallest root-cause fix, run targeted verification, and summarize root cause, changed files, tests, commands, and remaining uncertainty.`
